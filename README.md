# digital_me

## Installation:

Digital me is installed as part of Jumpscale [see installation instructions here](https://github.com/threefoldtech/jumpscale_core) 
## Running

you can run Digital_me using `js_shell`
```bash
js_shell 'j.servers.digitalme.start()'
```
